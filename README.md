INSTALLARE NODE JS

- Cercare "node version manager windows" su Google e cliccare su Github di Corey Butler (coreybutler/nvm-windows)
https://github.com/coreybutler/nvm-windows
- Sulla pagina, guardare a destra e cliccare su Releases, poi su Contributors selezionare " nvm-setup.exe".


O SEMPLICEMENTE ANDARE SULLA PAGINA UFFICIALE DI NODE JS 
- Cliccare "Get Node.js"
- Poi cliccare su "Windows Installer.msi"
